1. Install dependencies:

```bash

```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

Edit the content in the components to personalize:

- Update personal information in `Hero.tsx`
- Add your projects in `Portfolio.tsx`
- Modify skills and experience in `About.tsx`
- Add blog posts in `Blog.tsx`
